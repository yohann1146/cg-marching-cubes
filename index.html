Marching Cubes Algorithm

A C++ implementation for 3D model generation from medical imaging data.

Introduction

This project is an implementation of the Marching Cubes algorithm, a well-known technique in computer graphics for extracting a polygonal mesh of an isosurface from a 3D scalar field. The implementation is done in C++ and demonstrates the process of generating a 3D model from a medical imaging dataset.

The Marching Cubes Algorithm

The Marching Cubes algorithm processes a 3D grid of scalar values (voxels). For each cube (a cell of 8 neighboring voxels) in the grid, it determines how the isosurface intersects that cube. Based on which of the 8 vertices of the cube are inside or outside the surface (determined by comparing their values to a threshold), the algorithm looks up a corresponding polygonalization of the cube in a pre-calculated lookup table. By combining the polygons from all cubes, a complete 3D mesh of the surface is generated.

Dataset: MedMNIST 3D

The dataset used for this project is the MedMNIST 3D adrenal dataset. This is a collection of 3D medical images. For this project, a single image is extracted from the dataset, converted into a simple text format, and then used as the input for the Marching Cubes algorithm.

Output

The program will generate an output.off file. The .off (Object File Format) file contains the vertices and faces of the generated 3D mesh. This file can be viewed using 3D modeling software like MeshLab or Blender.
